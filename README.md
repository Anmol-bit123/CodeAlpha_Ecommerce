# ShopAlpha - E-commerce Store

A full-stack e-commerce application built as part of CodeAlpha Full Stack Development Internship.

## Features
- User Registration & Login (bcrypt password hashing)
- Product Listings & Detail Pages
- Shopping Cart
- Order Processing & Order History

## Tech Stack
- **Frontend:** HTML, CSS, EJS
- **Backend:** Node.js, Express.js
- **Database:** MySQL

## Setup
1. Clone the repo
2. Run `npm install`
3. Import `schema.sql` into MySQL
4. Update DB password in `index.js`
5. Run `node index.js`
6. Visit `http://localhost:8080`
